# 📚 StudySync

A web-based study productivity tool designed to help students stay organised and focused.

## ✨ Features

- ✅ **Task Tracking** — Create and manage your study to-do list
- ⏱️ **Pomodoro Timer** — Stay focused with timed study sessions
- 📝 **Notes & Flashcards** — Write notes and review with flashcards
- 📊 **Progress Tracking** — Monitor your study habits over time
